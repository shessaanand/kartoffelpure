# KartoffelPure

## Overview
Privacy-focused browser built in Rust.

## Stack
- Rust
- GTK4
- WebKitGTK
- SQLite
- Tokio

## Goals
- Fast
- Minimal
- Privacy-first
- Open source

## Features
- Tabs
- Bookmarks
- History
- Downloads
- Settings

## Non Goals
- Custom browser engine
- Electron
- Excessive dependencies
